## Requirements

-   Install Ubuntu version 20.04
-   Install [Node.js](https://nodejs.org/) version 22.2.0
-   Install [Yarn](https://classic.yarnpkg.com/) version 1.22.22

## Install dependencies

    $ yarn install --frozen-lockfile

## Build production

    $ yarn build:ext
